 1.甚麼是經濟學

筆記說明

此筆記用途在於國立屏東大學資訊管理系大三上經濟學重點整理
並非所有人都適用

如果有侵犯到版權的問題在聯繫我，再聯絡我，我在移除


經濟學的定義

  • 經濟學是一門研究個人、企業、政府與整個社會在面對稀少性與誘因時如何做選擇的一門社會科學
  • 經濟學的本質為稀少性(scarcity)
  • 價格是一種誘因
  • 經濟學有兩個主要領域
    • 個體經濟學
      • 研究個人與企業的選擇
    • 總體經濟學
      • 研究一國經濟與全球經濟的表現


兩大經濟問題

  • 生產甚麼、如何生產與為誰生產
    • 生產甚麼
      • 商品(有形)與服務(無形)
      • 不同國家在同一時點有不同的產品組合
      • 同一國家在不同時點會有不同的產品組合
    • 如何生產
      • 利用生產要素(factors if production)生產
        • 土地
        • 勞動
          • 勞動的品質決定於勞工的人力資本
        • 資本
          • 廠商用來生產商品與服務的工具、設備、機器、廠房與其他建築物稱
          • 錢、股票與債券為金融資本(無法用來生產，所以不是生產要素)
        • 企業才能
          • 企業才能：企業家組織勞動、土地與資本的能力
        • 全都是有限資源
      • 不同國家在同一時點有不同的產品組合
      • 同一國家在不同時點會有不同的產品組合
    • 為誰生產
      • 消費組合決定於所得，所得來自生產要素的報酬
        • 土地 => 地租(rent)
        • 勞動 => 工資(wages)
        • 資本 => 利息(interest)
        • 企業才能 => 利潤(profit)
      • 勞動賺最多
  • 人們為追求自利所做的選擇是否也提升了社會利益
      社會利益重要議題
    • 全球化(globalization)
    • 資訊時代獨佔(information-age monopoly)
    • 氣候變遷(climate change)
    • 金融不穩定(financial instability)


經濟學家的思考方式

  • 選擇是一種取捨(tradeoff)
  • 人們藉由比較利益(benefit)與成本(cost)做出理性的選擇(rational choice)
  • 利益是你從某物所獲取的利得
  • 成本是你獲取某物所必須放棄的(must give up)
  • 大多數的選擇是關於在邊緣(at margin)所做的多少("how-much")數量的選擇
  • 選擇隨誘因起舞
