 開啟攝影機擷取影像


鏡頭拍攝，當按下ESC鍵時，結束錄影並擷取影像

    import cv2
    cap = cv2.VideoCapture(0)
    
    fourcc = cv2.VideoWriter_fourcc(*'XVID')
    out = cv2.VideoWriter('output.avi', fourcc,20,(640,480))
    
    while(cap.isOpened()):
        ret, frame = cap.read()
        if ret == True:
            out.write(frame)
            cv2.imshow('frame', frame)
            key = cv2.waitKey(1)
            if key == 27:#ESC
                break
        else:
            break
    cap.release()
    out.release()
    cv2.destroyAllWindows()
